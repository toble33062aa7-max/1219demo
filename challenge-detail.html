挑戰規格明細
挑戰題目
智慧路口：即時交通影像辨識與分析解決方案
出題機關
臺北市政府交通局
徵件期間
114/12/15 ～ 115/01/30 (剩餘 42 天)
實證獎助金
新台幣 2,500,000 元 (含稅)
預計執行期程
決標後 6 個月內完成 POC 驗證
實證場域地點

主要場域：臺北市信義區基隆路與忠孝東路交叉口。

備用場域：內湖科學園區瑞光路沿線路口。

(註：需配合交通局安排現勘)

需求與規格

本計畫旨在解決尖峰時刻路口壅塞問題，徵求具備邊緣運算能力之 AI 影像辨識方案。

  • 辨識標的：需能區分大客車、小客車、機車、自行車及行人。
  • 準確率要求：日間辨識率需達 95% 以上，夜間需達 90% 以上。
  • 系統介接：需提供 RESTful API 介接本局交控中心，資料延遲不得超過 3 秒。
  • 硬體限制：需可安裝於現有號誌桿上，且具備 IP66 防水防塵等級。
參賽資格

1. 依公司法設立之本國公司，或外國公司在台分公司。

2. 新創團隊（成立未滿 5 年者尤佳）。

3. 最近 3 年無欠繳稅款紀錄。

評選方式
書面審查 (40%) + 簡報決選 (60%)。簡報需包含 Live Demo 展示。
聯絡窗口
林專員 (02-2720-8889 分機 1234)
