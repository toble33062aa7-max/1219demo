挑戰題目
智慧路口：即時交通影像辨識與分析解決方案
技術領域分類
AI 影像辨識 / 5G IoT
建議實證場域
臺北市信義區 (基隆路與忠孝東路口)
現況問題痛點
目前本市主要幹道於上下班尖峰時刻，常因固定號誌時相無法應對突發性車流，導致路口回堵嚴重。現有之感應線圈維護成本高且數據單一，缺乏車種辨識與人流分析能力，無法作為動態號誌調整之依據。
技術規格要求
1. 需利用路口現有 CCTV 影像進行分析，不額外架設大型硬體為原則。
2. 辨識準確率：日間 > 95%，夜間 > 85% (需包含大客車、小客車、機車、行人)。
3. 數據延遲：影像分析數據回傳至中控中心延遲需小於 2 秒。
4. 介接規範：需提供標準 RESTful API 介接本局交控中心平台。
評選與審查標準
技術創新性 (30%)、場域可行性 (30%)、團隊執行力 (20%)、經費編列合理性 (20%)。
預期實證效益
1. 降低路口尖峰時段平均延滯時間 15% 以上。
2. 節省人工車流調查人力成本約 50 萬元/年。
3. 建立即時交通戰情儀表板，提供交通決策輔助。
