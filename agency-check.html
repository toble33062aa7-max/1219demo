申請資格

2. 成立 8 年內之新創企業尤佳。
3. 具備過往智慧交通或影像分析相關實績者。
評選標準
技術創新性 (30%)、場域可行性 (30%)、團隊執行力 (20%)、經費編列合理性 (20%)
需求詳述
現況問題
目前本市主要幹道於上下班尖峰時刻，常因固定號誌時相無法應對突發性車流，導致路口回堵嚴重。現有之感應線圈維護成本高且數據單一，缺乏車種辨識與人流分析能力，無法作為動態號誌調整之依據。
技術規格
1. 需利用路口現有 CCTV 影像進行分析，不額外架設大型硬體。
2. 辨識準確率： 日間 > 95%，夜間 > 85% (需包含大客車、小客車、機車)。
3. 延遲時間： 影像分析數據回傳延遲需 < 2 秒。
4. 介接規範： 需提供 RESTful API 介接本局交控中心平台。
預期效益
1. 降低路口尖峰時段平均延滯時間 15% 以上。
2. 節省人工車流調查人力成本約 50 萬元/年。
3. 建立即時交通戰情儀表板，提供決策輔助。
